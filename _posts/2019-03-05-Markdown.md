### 白科技

| 写法   | 效果   | 描述   |
| --- | --- | --- |
| `~~中划线~~`   | ~~中划线~~   | 中划线 |
| `<u>下划线</u>`   | <u>下划线</u> | 下划线 |
| `表格内<br>换行` | 表格内<br>换行 | 表格内换行 |
|   |   |   |



### 黑科技

* 缩进

  * 写法：

    ```html
    I am head
    <div style="margin-left: 1.5rem;">
    I am Content
    </div>
    ```

  * 效果：

    I am head
    <div style="margin-left: 1.5rem;">
    I am Content
    </div>
----------

* 画图表：[文档](https://mermaidjs.github.io/usage.html)

  * 写法：

    ```
    <div class="mermaid">
    graph TD
    A[RecycleView] --> B(LayoutManager)
    A --> C(Item Animator)
    A --> D(Adapter)
    </div>
    ```

  * 效果：

    <div class="mermaid">
    graph TD
    A[RecycleView] --> B(LayoutManager)
    A --> C(Item Animator)
    A --> D(Adapter)
    </div>
