### CPU 指令集
大体上CPU指令集可以分为“复杂指令集”和“精简指令集”两大阵营

| 复杂指令集（CISC） | 精简指令集（RISC）       |
| ------------------ | ------------------------ |
| x86                | ARM（armeabi，v7a，v8a） |
| x86_64             | PowerPC                  |
| IA64               | MIPS                     |

### 公司

| Intel       | AMD    | ARM  | IBM     |
| ----------- | ------ | ---- | ------- |
| x86（8086） | x86    | ARM  | PowerPC |
| ~~iA64~~    | x86_64 |      |         |

### 现状

1. 先如今使用最多的都是 x86 以及 ARM 架构了，其中手机等移动设备一般都使用 ARM 架构，其他的一般都是 x86。
